# Toyota Product Knowledge Course

เรียนรูัผลิตภัณฑ์ Toyota Thailand แบบ Interactive — React + Vite

## วิธีใช้งาน

```bash
npm install
npm run dev       # dev server
npm run build     # build → dist/
npm run preview   # preview build
npm run deploy    # deploy GitHub Pages
```

## หมายเหตุ

ไฟล์ PDF ทั้งหมดดึงจากเซิร์ฟเวอร์ Toyota Thailand โดยตรง (`www.toyota.co.th/media/product/series/download/`) — ไม่ต้องมีไฟล์ PDF ใน Repo
